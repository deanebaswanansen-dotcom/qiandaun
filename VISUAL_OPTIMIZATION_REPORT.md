# qiandaun 前端视觉优化建议

## 定位

这是一个炫技型幻想 RPG 官网，视觉目标应是“电影级开场 + 高级互动宣传片 + 可截图传播的关键画面”。当前素材质量不错，但页面更像素材陈列，缺少统一导演感和视觉高潮。

## 核心问题

1. 首屏冲击力不足

   首屏有视频、有大标题、有章节切换，但视觉重心分散。建议把首屏改成电影开场：视频暗部压低，标题放大，副文案减少，保留一个强 CTA，让用户第一眼记住“龙与幻想”。

2. 视觉层级过平均

   多数模块都在使用发光、描边、模糊、阴影和动效，导致每个区域都想当主角。建议每个区域只保留一个标志性视觉手段：首屏负责震撼，世界观负责庄严，塔罗负责神秘，地图负责探索，角色负责情绪，图库负责展示。

3. 炫技点没有串成演出

   页面已有粒子、点击爆裂、卡牌翻转、地图地标、音频和视频，但它们是分散效果。建议设计一条连续演出线：进入页面 -> 标题浮现 -> 龙影掠过 -> 地图亮起 -> 卡牌翻面 -> 角色登场 -> 预约按钮收束发光。

4. 图片展示方式偏普通

   AI 原画质量较高，但图库仍是常规网格。建议改成全屏横向画廊、视差滚动、局部放大镜和电影级 lightbox，让每张图像艺术展海报一样被观看。

5. 中文排版不够史诗

   当前中英文混排有质感，但字号、行距、字重和段落节奏偏平均。建议减少大段说明，把世界观文字做成碑文、古籍页、任务日志、羊皮卷或章节铭文，强化幻想作品的仪式感。

6. 移动端缺少专属构图

   当前移动端主要是桌面布局压缩。建议把手机首屏当竖版预告片设计：标题、视频主体、章节切换、CTA 在一屏内形成完整构图，避免只是把桌面内容堆成单列。

## 优先级建议

第一优先级：重做首屏构图和动效节奏，让第一屏具备截图传播力。

第二优先级：统一视觉层级，每个模块只保留一个最强记忆点。

第三优先级：重做图库和地图交互，把静态展示升级成可探索体验。

第四优先级：优化中文排版和移动端专属视觉。

## 可落地改法

- 首屏：增加 poster 图，视频加暗角和景深层，标题减少到 1 行主标题 + 1 行短标语。
- 章节切换：从普通文字切换改为“镜头切片”，每次切换带背景色、粒子方向和标题文案同步变化。
- 塔罗区：翻牌后增加短暂停顿、光圈扩散和卡牌音效，形成仪式感。
- 地图区：点击地标时让地图局部发光，预览图同步换色，旁边显示地点档案。
- 角色区：角色卡从资料卡改成“同伴登场”，加入台词、签名式技能和情绪光效。
- 图库区：从网格改为沉浸式画廊，点击后进入全屏观赏模式。
- 移动端：单独设计 390px 宽度首屏，保留大标题、视频主体和一个核心按钮。

## 验收标准

- 电脑首屏停留 3 秒内能明确感受到“幻想 RPG 大作官网”。
- 手机首屏不用滚动即可看到标题、主体画面和核心按钮。
- 每个模块都有一个明确视觉记忆点，不能所有模块都使用同一套发光卡片。
- 图库点击后有全屏沉浸感，图片不只是普通弹窗。
- 页面动效有节奏变化，不能一直同强度闪烁和发光。
