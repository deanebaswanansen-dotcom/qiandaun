# 👑 龙与幻想 (Dragon Fantasy) · AAA级温馨西幻游戏官方宣传页

> **温馨西幻 / 少年启程 / 电影级沉浸视听 / 生成式音频合成 / 现代玻璃拟态美学**

本项目基于一段 6 秒首尾帧生成的 AI 动画视频，以前端高阶审美与底层工程架构，将其打造成了一座宛如《原神》、《塞尔达传说》或《最终幻想》般的高品质、多屏沉浸式 **AAA 级游戏官方网站 / 宣传发布页**。

## 🔥 本次重构与审美升级核心亮点

### 1. 🎵 完美解决“我声音呢？”：Web Audio API 实时合成视听引擎
为什么之前点击 **Sound On** 没有声音？因为工作区原本并没有放置 `bgm.mp3` 与 `ambience.mp3` 音频文件。
为了在不依赖外部第三方音频文件的前提下提供极致的视听沉浸感，我们在 [main.js](file:///C:/Users/PC/Desktop/qiandaun/main.js) 中创造性地编写了 **`FantasyAudioEngine` (生成式自适应网页合成音频引擎)**：
- **🍃 微风与落叶沙沙声**：采用布朗噪声 (Brownian Noise) 算法与低通滤波器 (`BiquadFilterNode`)，配合 LFO 低频振荡器，实时模拟大自然一阵阵吹拂的温柔秋风。
- **✨ 竖琴与风铃分解和弦**：基于西幻治愈系五声音阶 (C Major / G Major)，利用三角波振荡器 (`OscillatorNode`) 与包络衰减，每隔数百毫秒实时弹奏清澈甜美的竖琴泛音。
- **🎻 管弦背景和声底垫**：低频正弦波与三角波平滑铺底，营造温暖、怀念且充满希望的王国史诗氛围。
- **智能双重保障**：当您点击 **Sound On** 时，系统会优先尝试播放本地 MP3；一旦检测到文件缺失或加载失败，合成引擎会立刻无缝接管，为您实时演奏专属于这片大陆的交响诗篇！

### 2. 📜 完美解决“怎么往下滑？”：智能协同滚动与全方位官网叙事
之前为了将页面锁死在单一视频屏，设置了 `overflow: hidden` 并拦截了全局鼠标滚轮。现已解除限制并升级为 **智能协同交互模式**：
- **🎯 智能分流**：当您在首屏将鼠标悬停在左下角文案区或右侧章节时间轴时，滚轮滑动为**切换视频剧情章节**；当鼠标移至画面其他区域时，滚轮即可**顺畅向下滑动**，探索广阔的游戏官网世界！
- **👇 向下探索指示器**：首屏底部中央新增了动感呼吸箭头 **「向下探索 · ENTER THE REALM」**，点击或向下滑动即可平滑过渡至下文。

### 3. 🏰 殿堂级审美丰富：五大板块构建完整游戏 IP 宣传页
除首屏 6 秒循环序章外，以高级 UI 审美补充了四大全新功能分区，并使用 AI 生成了原画素材：
- **📍 第二屏：世界观 · 风起之地与金色诗篇 (World Lore)**
  - 采用古籍编年史排版与左右分栏设计。搭配 AI 生成的超高清 **白垩王城概念原画** ([concept_castle.jpg](file:///C:/Users/PC/Desktop/qiandaun/assets/images/concept_castle.jpg))，结合 3D 景深悬停光晕，向用户娓娓道来“晨曦平原”、“风车王城”与“少年罗盘”的动人背景。
- **🎮 第三屏：游戏特色 · 沉浸式温馨大世界 (Gameplay Features)**
  - 现代拟态毛玻璃 (Glassmorphism) 网格卡片布局。设计了 **「实时气候与光影」**、**「无缝治愈大世界」**、**「自适应管弦交响」** 三大特色展示块，鼠标悬停即触发卡片上浮与金色流光边框特效。
- **🧭 第四屏：旅途同伴 · 角色预览 (Character Showcase)**
  - 专为主角设计了角色档案卡。配以 AI 绘制的 **少年冒险者立绘** ([concept_character.jpg](file:///C:/Users/PC/Desktop/qiandaun/assets/images/concept_character.jpg))，展示其角色定位、故事台词（*“把故乡留在身后的金色风里”*）以及“微风护佑”、“星图指引”两大特色技能。
- **🎁 第五屏：全球公测预约与数据互通平台 (Pre-registration CTA)**
  - 极具仪式感的金色放射渐变视觉横幅。提供了纪念礼包预览（限定背饰「初风之翼」）、交互式手机/邮箱预约表单，以及 PC / iOS / Android / PS5 / 云游戏全平台支持徽章。
  - 附带 **实时庆祝弹窗 (Toast Modal)**：输入任意内容点击预约，即刻触发奢华的预约成功庆贺提示！
- **👑 底部：工作室铭牌与适龄提示 (Footer)**
